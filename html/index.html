Bigbluebutton Manager Installation Guid

Bigbluebutton Manager is a tool to manage your bigbluebutton server meetings.Its provide a unique GUI where you can create meeting,meeting room and can invite people without even login on BBB server.BBBmanager provide stats and reports as well.

Installation Guide

1. Prior installation make sure you have mysql database installed in your machine

2. Clone code from https://github.com/optimizium/bbbmanager

3. cd myconfec

4. Run pip install -r requirements.txt

5. Install apache2 by running sudo apt-get install apache2

6. Please refer link https://docs.djangoproject.com/en/1.7/howto/deployment/wsgi/modwsgi/ to configure apache with django.

7. Once apache configuration done you can go to http://YOUR_BASE_URL/account/login

Setting up development envronment.

1. Clone code from https://github.com/optimizium/bbbmanager

2. cd myconfec

3. Run pip install -r requirements.txt

4. Now run python manage.py runserver

5. You should be able to access o your local url
